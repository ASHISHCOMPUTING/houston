# Architecture
* HTML/CSS/JS/Node.js
* GitHub
* Jenkins
* Mongo DB
* Stripe Connect

For a local development environment, you will need:
* `nodejs-legacy`
* `mongodb`
* `npm`

run `npm install` to grab dependencies
